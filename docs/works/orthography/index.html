design choices

the author wanted to distinguish any phonological difference. the system distinguishes [≈ìÀê, …µ] like jyutping ‚Äπoe, eo‚Ä∫, not like cantonese pinyin ‚Äπoe‚Ä∫. the system also distinguishes lengths of the other vowels, which is useful for beginners, while left optional as they have no ambiguity and is not necessary for natives, except for [aÀê, …ê].

the author tried to make the system minimal. the system adopts cantonese pinyin ‚Äπy‚Ä∫, not jyutping ‚Äπyu‚Ä∫, and adopts jyutping ‚Äπz, c‚Ä∫, not cantonese pinyin ‚Äπdz, ts‚Ä∫. the system uses many diacritics in order to avoid numerals and doubled letters, which are ugly in the autohr's opinion, while providing the ASCII alternative easy to type.

the system replaces jyutping ‚Äπw‚Ä∫ with ‚Äπv‚Ä∫, as ‚Äπv‚Ä∫ is the primitive latin letter for [w] sound and cantonese does not need to distinguish [w, v]. the author does not like digraphs like ‚Äπng, oe‚Ä∫, which breaks uniformity. as we have ‚Äπq, w‚Ä∫ unused, we can avoid them.
