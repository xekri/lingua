# polish morpho-phonological orthography proposal

## new letters

|        | old       | new      |
| :----: | :-------- | :------- |
|  [h]   | ch        | h        |
|  [ts]  | c         | ķ, ț     |
|  [dz]  | dz        | ģ, ḑ     |
| [s, ɕ] | s         | s, š, ȟj |
| [z, ʑ] | z         | z, ǧ     |
|  [tʂ]  | cz        | ǩ, ť     |
|  [dʐ]  | dż        | ď        |
|  [ʐ]   | ż         | ǧ, ž     |
|  [ʂ]   | sz        | ȟ, š     |
|  [e]   | e         | e, ė     |
|  [je]  | (C)ie, je | ě        |
|   ∅    |           | ė, ë     |

assume `i=jy, ě=je`
